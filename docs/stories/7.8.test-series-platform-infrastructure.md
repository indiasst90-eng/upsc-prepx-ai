---

## Story

**As a** UPSC aspirant,
**I want** a comprehensive test series platform for full-length mock tests,
**so that** I can practice complete papers in exam-like conditions.

---

## Acceptance Criteria

1. Database schema: `test_series` table with test_id, title, type (Prelims/Mains), duration, total_marks, instructions
2. Test types: Prelims (100 MCQs, 2 hours), Mains GS (20 questions, 3 hours), Essay (2 topics, 3 hours)
3. UI page: /practice/tests displays available tests with filters (type, difficulty, completed/pending)
4. Test enrollment: users click "Start Test" to begin, triggers timer and locks test interface
5. Full-screen mode: test runs in distraction-free full-screen with timer countdown
6. Question navigation: numbered grid showing attempted/unattempted/marked-for-review questions
7. Auto-submit: test auto-submits when timer expires, saves all answers
8. Database: `test_attempts` table with user_id, test_id, started_at, submitted_at, answers_json, score
9. Test scheduling: admins can schedule tests to go live at specific dates/times
10. Entitlement: Free users get 1 mock test, Pro users get unlimited tests

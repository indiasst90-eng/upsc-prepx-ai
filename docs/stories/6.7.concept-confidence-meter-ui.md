---

## Story

**As a** UPSC aspirant,
**I want** a visual confidence meter showing my mastery level for each topic,
**so that** I can quickly identify which topics I've mastered and which need more work.

---

## Acceptance Criteria

1. Confidence scoring: calculates 0-100 confidence score per topic using quiz performance (40%), time spent (20%), video completion (20%), answer writing grades (20%)
2. Color coding: Red (0-40), Yellow (41-70), Green (71-100)
3. UI: /confidence page displays all syllabus topics with circular progress meters
4. Filterable: filter by subject (Polity, History, Geography, etc.), confidence level, priority
5. Sortable: sort by confidence score (ascending/descending), last updated, topic name
6. Trend visualization: shows confidence delta (arrow up/down) compared to last week
7. Database: `topic_confidence` table with user_id, topic_id, confidence_score, last_updated, trend
8. Suggested actions: for red topics, shows "Start Learning" button; for yellow, shows "Practice More"; for green, shows "Maintain"
9. Confidence report: weekly Revideo video summary of top 5 confidence gains and top 5 concerns
10. Gamification: badges for reaching 90+ confidence on 10, 25, 50, 100 topics

# Story 7.4: Answer Writing - Model Answer Database

**Epic:** 7 - Practice & Evaluation - Answer Writing & Essays
**Story Number:** 7.4
**Status:** Draft
**Created:** December 23, 2025
**Sprint:** Sprint 8 (Epic 7)
**Estimated Effort:** Medium (M)
**Risk Level:** LOW

---

## Story

**As a** UPSC aspirant,
**I want** access to model answers for every question,
**so that** I can compare my response with ideal answers and learn the expected format.

---

## Acceptance Criteria

1. Database: `model_answers` table with question_id, answer_text, word_count, score_breakdown, author, source
2. Answer sources: (1) AI-generated model answers, (2) Topper answers (curated), (3) Admin-uploaded answers
3. AI generation: for questions without model answers, generate using prompt "Write a perfect 250-word UPSC Mains answer for: {question}"
4. Quality control: model answers must score 35+/40 on evaluation engine before being published
5. UI: "View Model Answer" button visible after user submits their answer
6. Comparison mode: split-screen view showing user answer on left, model answer on right
7. Highlighting: highlights differences in structure, keywords, examples used
8. Download: users can download model answers as PDF
9. Admin interface: /admin/model-answers allows admins to upload, edit, approve answers
10. Versioning: tracks multiple versions of model answers, users see highest-rated version

**Risk Level:** LOW

---

## Story

**As a** UPSC aspirant,
**I want** to browse and search the complete PYQ database,
**so that** I can find questions by year, subject, topic, or difficulty.

---

## Acceptance Criteria

1. UI page: /pyqs displays comprehensive PYQ database with grid/list view toggle
2. Filters: year range (2010-2024), paper type (Prelims/Mains/Essay), subject, topic, difficulty
3. Search: full-text search across question text with highlighting
4. Sort options: by year (newest/oldest), relevance, difficulty, popularity (most viewed)
5. Question card: displays question text (truncated), year, paper type, subject tags, "View Explanation" button
6. Statistics: shows total questions, year-wise distribution, subject-wise breakdown
7. Bookmarking: users can bookmark questions for later review
8. Practice mode: "Start Practice Session" button selects random 10 questions matching filters
9. Database query: uses Supabase indexes on year, paper_type, subject for fast filtering
10. Pagination: loads 20 questions per page with infinite scroll
